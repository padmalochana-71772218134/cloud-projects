# 📦 Cloud Chat Lite

**Cloud Chat Lite** is a real-time private chat application built with **HTML**, **CSS**, **JavaScript**, and **Supabase**. It supports user authentication, profile management, contact listing, file sharing, and private messaging between users.

---

## ✅ Features

### 🔐 Authentication

* Email/password-based **user registration** and **login**.
* **Session management** using Supabase's built-in auth system.
* **Logout** support to safely end sessions.

### 👤 User Profile

* Each user has a profile with:

  * **Username**
  * **Avatar URL**
* Profile is automatically created in the database on first login.
* Users can update their display name and avatar.

### 📇 Contact List

* Shows all **other registered users** except the currently logged-in user.
* Selecting a contact opens a dedicated **private chat**.

### 💬 Private Messaging

* 1-on-1 text messaging between users.
* Attach and send **files (images, docs, etc.)** via Supabase Storage.
* Messages display:

  * **Who sent it** ("You" vs. "Them")
  * **Text**
  * **Clickable download links** for file attachments.

## 🧰 Tech Stack

* **Frontend**: HTML, CSS, Vanilla JavaScript
* **Backend (BaaS)**: [Supabase](https://supabase.com/)

  * Authentication
  * Realtime Database
  * Storage (for files)

## 🛠 Setup Instructions

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/cloud-chat-lite.git
   ```

2. Open `supabaseClient.js` and set your `supabaseUrl` and `supabaseKey`.

3. Open `index.html` in your browser. You’re ready to chat!
